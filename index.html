How it Works (Phase I Demo):

  • This demo fetches news from **The Himalayan Times RSS feeds** via a **local backend server**.
  • The backend uses **VADER sentiment analysis** and serves only the **top 5 most positive headlines**.
  • You can click "Read Aloud" to hear an article's summary.
  • **New!** Click "Read Headlines" to hear all displayed headlines.
  • IMPORTANT: You must run the Python backend server (app.py) locally for this to work!
